# LittleDataBase_Local
    目前该版本运行于windows。
# other
    （１）目前支持插入，删除功能．
    （２）存储由Ｂ树实现，其中为了节省内存占用改为使用链表存储各个数据．
    （３）可以通过server命令将当前程式设置为远程端口，并在另一个程式内使用client命令登录该server端．目前支持
            １对１登录和访问．
    （４）支持添加和删除表．
    （５）未支持数据持久化．
